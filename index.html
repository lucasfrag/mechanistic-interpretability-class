O que uma rede guarda nas suas ativações — e por que o significado não mora nos neurônios, e sim nas direções.

intuição
Bloco 1 · features

O significado está na direção

Uma ativação é um vetor. Um número isolado — um neurônio — quase nunca corresponde a um conceito limpo.

Feature = uma direção interpretável no espaço de ativação.

Andar nessa direção = ativar mais aquele conceito.

Se conceitos são direções, então combinar conceitos vira aritmética de vetores. É o que veremos a seguir.

A hipótese da representação linear — a base de quase tudo que vem depois.

Bloco 1 · direções se combinam

Aritmética de conceitos

Cada palavra é uma posição; cada deslocamento entre elas é uma direção — uma feature. Partindo de rei, subtraio a direção homem, somo mulher, e chego em rainha.

clique para andar pela cadeia, passo a passo
Analogias vetoriais (Mikolov et al., 2013) — a intuição vale para as features internas de um LLM.

Bloco 1 · o problema que aparece

Conceitos demais,
dimensões de menos

Direções ortogonais não colidem — mas só cabem d delas num espaço de d dimensões.

Um modelo conhece muito mais conceitos do que tem dimensões.

Milhares de features, centenas de dimensões. Como todas cabem?

Bloco 2

Superposição: features
além das dimensões

Se não cabem ortogonais, a rede as empacota em ângulos quase-perpendiculares — e aceita um pouco de interferência em troca.

interativoachado
Bloco 2 · a geometria da superposição

Features viram formas geométricas

Para minimizar interferência, N features espalham-se o mais longe possível umas das outras — nos vértices de um polígono regular.

—

2 features → linha (antípodas, 180°)
4 features → quadrado (90°, ortogonais!)
5+ features → pentágono, hexágono… (< 90°, interferência)
arraste para adicionar features e ver a forma mudar
Elhage et al. (2022), Toy Models of Superposition — features formam polígonos/politopos regulares.
09
interativo
Bloco 2 · o custo

O preço é a interferência

Duas features não perpendiculares “vazam” uma na outra. Essa medida de vazamento é o produto interno entre suas direções.

cos 90° = 0 → ortogonais, interferência zero (ideal)
cos 0° = 1 → mesma direção, interferência máxima (evitada)
—
A superposição empacota features em ângulos próximos de 90°. Fonte: Elhage et al. (2022).
10
Bloco 2 · a consequência

Um neurônio ≠ uma feature

Se as features se sobrepõem, um neurônio isolado responde a várias features não relacionadas ao mesmo tempo.

  • Isso é polissemanticidade — olhar um neurônio não revela um conceito limpo.
  • A feature verdadeira está espalhada por muitos neurônios, e cada neurônio mistura muitas features.

Então, como desembaralhar as features de volta?
